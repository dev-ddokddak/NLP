1. 자연어 처리 텍스트 분석 절차
    Q. 내가 직접 인스타그램에서 "보헤미안 랩소디" 해시태그관련 정보를 수집해서 시장반응을 분석해야 한다면 어떻게 할 것인가?
        1. 데이터 수집 단계: 인스타그램에서 "#보헤미안랩소디" 해시태그를 입력하고 포스트 검색결과를 수집
        2. 텍스트 전처리 단계: 포스트 내용을 일관된 포맷으로 정리
        3. 텍스트 분석 단계: 각 포스트 내용을 읽어보고 핵심 키워드, 긍정/부정/중립을 판단
        4. 시각화 단계: 정리한 내용을 보고서로 정리

    1) 데이터 수집
        - 데이터 수집(Data Collection)
            - 필요한 데이터를 선별하고 수집하여 저장하는 것
        - 데이터 정제(Data Cleansing)
            - 데이터를 쉽게 사용할 수 있도록 불필요한 부분을 제거
    2) 텍스트 전처리 단계
    - 텍스트 분석을 위해서 기계가 텍스트를 이해할 수 있도록 표준화하는 단계
        - 토큰화(Tokenizing)
            - 문장을 형태소로 분리하는 작업
        - 품사 부착
        - 원형 복원
        - 불용어 처리
    3) 텍스트 분석 단계
        - 주제어 찾기
            - 문서 내에서 주제를 발견하기 위한 모델
        - 문서 요약
            - 문서 내에서 주요 문장을 찾아 요약
        - 문서 분류
            - 문서 내 단어 혹은 문장을 분석하여 문서를 분류
        - 감성 분석
            - 문서 내 나타난 사람들의 태도, 의견, 성향 같은 주관성을 분석
    4) 시각화 단계
        - 시각화: 데이터 분석 결과를 쉽게 이해할 수 있도록 시각적으로 표현하고 전달하는 과정
