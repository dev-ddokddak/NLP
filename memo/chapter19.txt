TextRank 활용 핵심 키워드 추출
    - TextRank 개념:
        - 그래프 기반 랭킹 모델로 Google PageRank에 기반.
        - 텍스트 내 단어의 중요도를 그래프를 통해 판단.
    - 작동 원리:
        - 각 단어를 그래프의 노드로 처리하고, 단어 간 연결을 엣지로 표현.
        - 노드 간의 연결(엣지)는 단어의 동시 출현 빈도에 근거.
        - 각 노드에 PageRank 알고리즘 적용하여 중요도 계산.

    - TextRank 적용 과정:
        1. 텍스트 단위(단어)를 노드로 정의.
        2. 단어 간 관계를 엣지로 표현 (방향성 및 가중치 고려).
        3. TextRank 알고리즘을 이용해 각 노드의 중요도 계산.
        4. 중요도에 따라 키워드 순위화.
    - TextRank와 Google PageRank의 비교:
        - Google PageRank는 웹 페이지의 중요성 판단에 사용.
        - TextRank는 텍스트 내 단어의 중요도를 평가하는 데 사용.
    - 키워드 추출:
        - 텍스트에서 중요 단어 자동 식별.
        - 비지도 학습 방법론으로 문장 추출과 키워드 추출에 적용 가능.

    TextRank 알고리즘
        - 방법:
            - 단어 노드와 그들 사이의 관계를 고려하여 중요한 단어 식별.
            - 각 노드의 TextRank 점수 계산.
            - 노드 간의 관계에 기반한 가중치 행렬을 통해 점수 계산.
        - 계산 완료:
            - 최종적으로 각 노드(단어)의 TextRank 점수를 통해 중요 단어 도출.
    - 결론
        - TextRank는 자연어 처리에서 핵심 키워드 추출에 효과적인 방법.
        - 깊은 언어 지식이나 특정 도메인 코퍼스 필요 없이 다양한 텍스트에 적용 가능.
